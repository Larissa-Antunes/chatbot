# Chatbot com IA


* Chatbot desenvolvido em Python, utilizando a IA do Gemini para oferecer respostas mais precisas e interativas.

## ⚙️ Configuração do Ambiente
### Criando e Ativando o Ambiente Virtual

**Windows:**
```bash
python -m venv chatbot chatbot\Scripts\activate
````

### Instalação das Bibliotecas

```
    pip install numpy
    pip install tiktoken
    pip install flask
    pip install opencv-python
    pip install google-generativeai
    pip install python-dotenv  
```
